A fun and interactive Simon Says memory game built using simple web technologies. 
Test your memory skills by repeating the sequence of colors in the correct order!

Features
Color-based interactive UI
Increasing difficulty with each level.
Game over detection with restart option
Smooth animations and responsive design

Technologies Used
HTML – Structure of the game
CSS – Styling and animations
JavaScript – Game logic and interactivity

How to Play
Press any key (or click Start) to begin the game
Watch the sequence of colors shown
Repeat the sequence by clicking the buttons
Each round adds a new color to the sequence
If you click the wrong color → Game Over
